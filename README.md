# CodSoft Python Internship - Task 4: Rock-Paper-Scissors Game

## Description
A command-line based Rock-Paper-Scissors game built using Python. 
The user plays against the computer, which makes a random choice. 
The program determines the winner based on standard game rules and 
keeps track of the score across multiple rounds.

## Features
- User can choose Rock, Paper, or Scissors
- Computer generates a random choice using Python's `random` module
- Automatically determines the winner (User, Computer, or Tie)
- Tracks and displays cumulative scores for both User and Computer
- Handles invalid input gracefully
- Allows the user to replay multiple rounds

## Technologies Used
- Python 3

## How to Run
1. Clone this repository
2. Run the file using: `python codsoftTask4.py`
3. Enter your choice (Rock/Paper/Scissors) when prompted
4. View the result and running score
5. Choose to play again or exit
